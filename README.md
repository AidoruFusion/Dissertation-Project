## Dataset

The `data/` folder is **not included** in this repository due to file size constraints.

You must create it manually and download the required files before running any scripts.

**Steps:**

1. Create the `data/` folder in the project root:
   ```
   mkdir data
   ```

2. Download the dataset from:
   https://drive.google.com/drive/folders/1xDoFBiXHJ-02An15YkpIXCybQVCBRE21?usp=sharing

3. Place the downloaded files inside the `data/` folder.
